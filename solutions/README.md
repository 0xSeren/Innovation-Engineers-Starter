# Solutions (instructor branch)

⚠️ **Spoilers.** These are worked answers to every exercise — one valid
implementation each, with the `TODO` blocks filled in. They live only on the
`instructor` branch, not on `main` (the student starter). Tuning constants
(`Kp`, `MS_PER_CM`, `GYRO_SIGN`, `CELL_CM`, …) are reasonable defaults and still
need calibrating per robot.

Every solution compiles for `arduino:avr:uno`. Layout mirrors the exercises:

```
solutions/DayN_.../ExNN_Name/ExNN_Name.ino
```

For the navigation exercises there are also fuller standalone reference
programs outside this repo at `~/projects/code/`:
`LineFollower`, `Bug1`, `Bug2`, `MazeSolverLSRB`, `MazeSolverTremaux`.
